1. Clone the repository
2. Configure EmailJS:
   - Sign up at [EmailJS](https://www.emailjs.com/)
   - Replace `YOUR_PUBLIC_KEY`, `YOUR_SERVICE_ID`, and `YOUR_TEMPLATE_ID` in script.js
3. Open index.html in a browser or use a local server
